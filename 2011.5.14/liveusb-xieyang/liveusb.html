LiveUSB

Defined:
  1. live USB is a USB flash drive containing a operating system which can be booted.
  2. Live USBs are closely related to live CDs, but ability to save settings permanently
  3. For system administration, data recovery without a permanent installation on the local hard disk drive.

U盘

Defined:

Universal Serial Bus
组成:
  1. Male type-A USB connector – interface
  2. USB mass storage controller – store configureation
  3. NAND flash memory chip – store data
  4. Crystal oscillator – produces the device's main 12 MHz clock signal and controls the device's data output through a phase-locked loop.

bootloader

Function:

load the operating system properly
Example:
  1. GRUB
  2. BOOTMGR
  3. Syslinux
  4. LILO
  5. NTLDR

computer boot step

Step:
  1. POST (Power On Self Test)
  2. Load MBR in to 0000:7C00 or 07C0:0000
  3. find bootloader and executes it
  4. load kernel

制作LiveUSB

Step:
  1. install MBR into USB Driver
  2. configure bootloader
  3. copy system files
